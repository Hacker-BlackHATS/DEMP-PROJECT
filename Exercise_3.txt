• Create a new view method called about which returns the following HttpResponse : 'Rango says here is the about page.'

ANS__ : 

In views.py
	
def about(request):
    return HttpResponse("Rango says here is the about page.")

• Map this view to /rango/about/ . For this step, you’ll only need to edit the urls.py of the Rango app. Remember the /rango/ part 	
	is handled by the projects urls.py .

ANS__ :

In urls.py file of the app

urlpatterns = [
    url(r'^$', views.index, name='index'),
    url(r'^about/', views.about, name='about'),
]
